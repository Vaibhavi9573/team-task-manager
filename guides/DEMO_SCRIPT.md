# Demo Video Script (2-5 minutes)

## 📹 Demo Video Tips

- **Length**: 2-5 minutes (focus on key features)
- **Format**: MP4, 720p minimum
- **Audio**: Clear narration or subtitles
- **Recording**: Use OBS Studio (free) or ScreenFlow (Mac)

## Script Outline

### Opening (15 seconds)
```
"Hi! I'm demonstrating the Team Task Manager application, 
a full-stack web app for project and task management with 
role-based access control. It's built with React, Node.js, 
and PostgreSQL."
```

### Demo Flow (2-3 minutes)

#### 1. Authentication (20 sec)
- Show login page
- Demo user signup
- Show dashboard after login
- Show logout functionality
```
"First, let's create an account and log in. 
The app uses server-side session tokens for secure access."
```

#### 2. Dashboard Overview (20 sec)
- Show dashboard stats (tasks, completed, overdue)
- Show projects list
- Show assigned tasks
```
"The dashboard gives you an overview of your work:
- Total tasks assigned to you
- Completed tasks
- Overdue tasks
- All your projects and tasks"
```

#### 3. Project Management (30 sec)
- Create new project
- Show project details
- Add team member
- Show project statistics
```
"Let's create a new project and add team members. 
Each project has:
- Members with different roles (Admin/Member)
- Real-time statistics
- Task overview"
```

#### 4. Task Management (30 sec)
- Create new task
- Assign to team member
- Set priority and due date
- Update task status
- Show status filtering
```
"Tasks can be created, assigned to team members,
and tracked through different statuses:
- To Do (not started)
- In Progress (being worked on)
- Done (completed)

Tasks can have priority levels and due dates."
```

#### 5. Advanced Features (20 sec)
- Show overdue tasks highlighting
- Show role-based permissions
- Show task filtering
```
"The app includes:
- Automatic overdue task highlighting
- Role-based access control
- Task filtering and sorting
- Real-time updates"
```

#### 6. Closing (15 sec)
```
"The application is fully responsive, supports team 
collaboration, and is deployed on Railway with a 
PostgreSQL database. All code is available on GitHub."
```

## Recording Steps

### Windows/Linux
1. Download OBS Studio (https://obsproject.com/)
2. Set resolution to 1280x720 or higher
3. Add browser window as source
4. Start recording
5. Perform demo actions
6. Stop recording
7. Export as MP4

### Mac
1. Use ScreenFlow or QuickTime
2. QuickTime: File → New Screen Recording
3. Select area and record
4. Export as MP4

## Editing Tips

- **Trim**: Remove unnecessary pauses
- **Add captions**: For key features
- **Speed up**: Slow parts (2x speed)
- **BGM**: Add background music (optional, copyright-free)
- **Transitions**: Simple cuts work best

## Export Settings

```
Format: MP4
Resolution: 1920x1080 (1080p) or 1280x720 (720p)
Bitrate: 5000-8000 kbps
FPS: 30
Audio: 128 kbps AAC
```

## Upload

1. **YouTube**: 
   - Sign in to YouTube Studio
   - Click "Create" → "Upload videos"
   - Select MP4 file
   - Add title and description
   - Add GitHub link in description
   - Set to "Unlisted" or "Public"

2. **GitHub Releases**:
   - Create release on GitHub
   - Upload MP4 as release asset
   - Link in README

3. **Google Drive**:
   - Upload MP4
   - Share link with access
