## 🆘 Help & Troubleshooting

| Problem | Solution |
|---|---|
| PostgreSQL won't start | Check Windows Services, or reinstall |
| Can't connect to database | Verify DATABASE_URL in .env is correct |
| Port 5000 in use | Change PORT in backend/.env to 5001 |
| Frontend can't reach API | Check FRONTEND_URL in backend/.env |
| CORS errors | Restart backend, clear browser cache |
| Deployment failed | Check Railway dashboard logs |
| API returning 401 | Check backend is running and sessions table exists |

**Detailed help**: See SETUP_GUIDE.md → Common Issues section
